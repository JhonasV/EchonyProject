# Red Social 

<h2>Hecho con / Made With </h2>
<ul>
  <li>ASP.NET CORE 2.0</li>
  <li>Razor Pages</li>

# Features 
* Login/Register
* Register IMAIL Confirmation
* Posts
* Friend request
* Users search bar
* Comments
* Comments Reply
* Profile picture
* Posts Likes
* Who liked the post
* Visit yours friends profile
